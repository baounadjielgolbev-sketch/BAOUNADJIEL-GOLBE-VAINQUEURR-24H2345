# Application de suivi et d'analyse

## Lancement
```bash
python -m streamlit run app.py
```

## Description
Application de suivi et d'analyse pour jardin communautaire urbain.

## Lien
https://github.com/A7m3dM0sta/ecogarden-tracker

## Fonctionnalités

- Suivi des récoltes
- Suivi de l'eau consommée
- Suivi du bénévolat
- Analyses descriptives
- Export des donnéess
